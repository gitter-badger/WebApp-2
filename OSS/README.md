# Aiursoft OSS Service
OSS Service for Aiursoft

## How to run

You need to install [Dot Net Core SDK](https://github.com/dotnet/core/tree/master/release-notes)

And excute following commands to run the project:

	dotnet restore
	dotnet run

That's it!
